详细信息

QDesignerPropertySheetExtension类允许您操作时显示在Qt设计师的属性编辑器窗口组件的属性。

QDesignerPropertySheetExtension提供了一组函数，这些函数通常用于查询组件的属性，以及在属性编辑器中操作属性的外观。例如：


  QDesignerPropertySheetExtension *propertySheet  = 0;
  QExtensionManager manager = formEditor->extensionManager();

  propertySheet = qt_extension<QDesignerPropertySheetExtension*>(manager, widget);
  int index = propertySheet->indexOf(QLatin1String("margin"));

  propertySheet->setProperty(index, 10);
  propertySheet->setChanged(index, true);

  delete propertySheet;

请注意，如果使用QDesignerPropertySheetExtension::setProperty () 函数更改属性的值，则不会更新撤消堆栈。为了确保可以使用撤消堆栈恢复属性的值，您必须使用QDesignerFormWindowCursorInterface::setProperty () 函数或其伙伴setWidgetProperty()代替。

在实现自定义组件插件时，QDesignerCustomWidgetInterface::initialize () 函数的参数提供了指向Qt 设计师当前QDesignerFormEditorInterface对象（在上面的示例中的formEditor）的指针。

可以通过使用 qt_extension() 函数查询Qt 设计师的扩展管理器来检索属性表或任何其他扩展。当你想释放扩展时，你只需要删除指针。

所有组件都有一个默认属性表，它用组件的属性（即用Q_PROPERTY () 宏定义的属性）填充Qt 设计师的属性编辑器。但是QDesignerPropertySheetExtension还提供了一个用于创建自定义属性表扩展的接口。

警告： Qt 设计师使用QDesignerPropertySheetExtension来提供其属性编辑器。每当在其工作区中选择组件时，Qt 设计师将查询组件的属性表扩展。如果所选组件具有已实现的属性表扩展，则此扩展将覆盖默认属性表。

警告：某些属性的属性表使用的数据类型是包含附加信息的不透明自定义QVariant类型，而不是普通的 Qt 数据类型。例如，枚举、标志、图标、像素图和字符串就是这种情况。

要创建属性表扩展，您的扩展类必须从QObject和QDesignerPropertySheetExtension继承。然后，由于我们正在实现一个接口，我们必须确保使用Q_INTERFACES () 宏让元对象系统知道它：


  class MyPropertySheetExtension : public QObject,
          public QDesignerPropertySheetExtension
  {
      Q_OBJECT
      Q_INTERFACES(QDesignerPropertySheetExtension)

  public:
      ...
  }

这使得Qt 设计师能够使用qobject_cast () 来获取QObject指针查询支持的接口。

在Qt 设计师 中，除非需要扩展，否则不会创建扩展。因此，在实现属性表扩展时，您还必须创建一个QExtensionFactory，即能够创建扩展实例的类，并使用Qt 设计师的扩展管理器注册它。

当需要属性表扩展时，Qt 设计师的扩展管理器将运行其所有注册工厂，为每个调用QExtensionFactory::createExtension () 直到找到第一个能够为所选组件创建属性表扩展的工厂. 然后这个工厂将创建扩展的一个实例。如果找不到这样的工厂，Qt 设计师将使用默认的属性表。

Qt 设计师 中有四种可用的扩展类型：QDesignerContainerExtension、QDesignerMemberSheetExtension、QDesignerPropertySheetExtension和QDesignerTaskMenuExtension。无论请求的扩展与多页容器、成员表、属性表还是任务菜单相关联，Qt 设计师 的行为都是相同的。

所述QExtensionFactory类提供的标准扩展工厂，并且还可以用作用于定义扩展工厂的接口。您可以创建一个新的QExtensionFactory并重新实现QExtensionFactory::createExtension () 函数。例如：


  QObject *ANewExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      if (iid != Q_TYPEID(QDesignerPropertySheetExtension))
          return 0;

      if (MyCustomWidget *widget = qobject_cast<MyCustomWidget*>
             (object))
          return new MyPropertySheetExtension(widget, parent);

      return 0;
  }

或者您可以使用现有工厂，扩展QExtensionFactory::createExtension () 函数，使工厂也能够创建属性表扩展扩展。例如：


  QObject *AGeneralExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      MyCustomWidget *widget = qobject_cast<MyCustomWidget*>(object);

      if (widget && (iid == Q_TYPEID(QDesignerTaskMenuExtension))) {
          return new MyTaskMenuExtension(widget, parent);

      } else if (widget && (iid == Q_TYPEID(QDesignerPropertySheetExtension))) {
          return new MyPropertySheetExtension(widget, parent);

      } else {
          return 0;
      }
  }

有关使用扩展类的完整示例，请参阅任务菜单扩展示例。该示例展示了如何为 Qt 设计师 创建自定义组件插件，以及如何使用QDesignerTaskMenuExtension类将自定义项添加到Qt 设计师的任务菜单。

请参考 QDesignerDynamicPropertySheetExtension, QExtensionFactory, QExtensionManager, 和创建自定义组件扩展。

成员函数文档

[virtual] QDesignerPropertySheetExtension::~QDesignerPropertySheetExtension()

销毁属性表扩展。

[pure virtual] int QDesignerPropertySheetExtension::count() const

返回所选组件的属性数。

[pure virtual] bool QDesignerPropertySheetExtension::hasReset(int index) const

如果给定index（索引）处的属性在Qt 设计师的属性编辑器中具有重置按钮，则返回 true ，否则返回 false。

请参考 indexOf() 和 reset()。

[pure virtual] int QDesignerPropertySheetExtension::indexOf(const QString &name) const

返回给定属性name（名称）的索引。

请参考 propertyName()。

[pure virtual] bool QDesignerPropertySheetExtension::isAttribute(int index) const

如果给定index（索引）处的属性是一个属性，则返回 true，该属性将从 UI 文件中排除，否则返回 false。

请参考 indexOf() 和 setAttribute()。

[pure virtual] bool QDesignerPropertySheetExtension::isChanged(int index) const

如果给定index（索引）处的属性值与属性的默认值不同，则返回 true ，否则返回 false。

请参考 indexOf(), setChanged(), 和 reset()。

[virtual] bool QDesignerPropertySheetExtension::isEnabled(int index) const

如果在Qt 设计师的属性编辑器中启用了给定index（索引）处的属性，则返回 true ，否则返回 false。

这个函数是在 Qt 5.0 中引入的。

请参考 indexOf()。

[pure virtual] bool QDesignerPropertySheetExtension::isVisible(int index) const

如果给定index（索引）处的属性在Qt 设计师的属性编辑器中可见，则返回 true ，否则返回 false。

请参考indexOf ()和setVisible ()。

[pure virtual] QVariant QDesignerPropertySheetExtension::property(int index) const

返回给定index（索引）处的属性值。

请参考 indexOf(), setProperty(), 和propertyGroup()。

[pure virtual] QString QDesignerPropertySheetExtension::propertyGroup(int index) const

返回给定index（索引）处的属性的属性组。

Qt 设计师的属性编辑器支持属性组，即相关属性的部分。可以使用setPropertyGroup () 函数将属性与组相关联。任何属性的默认组是定义它的类的名称。例如，QObject::objectName属性出现在QObject属性组中。

请参考 indexOf() 和 setPropertyGroup()。

[pure virtual] QString QDesignerPropertySheetExtension::propertyName(int index) const

回给定index（索引）处的属性名称。

请参考 indexOf()。

[pure virtual] bool QDesignerPropertySheetExtension::reset(int index)

将给定index（索引）处的属性值重置为默认值。如果可以找到默认值，则返回 true，否则返回 false。

请参考 indexOf(), hasReset(), 和 isChanged()。

[pure virtual] void QDesignerPropertySheetExtension::setAttribute(int index, bool attribute)

如果attribute为true，则给定index（索引）处的属性将成为将从UI 文件中排除的属性；否则将被包括在内。

请参考 indexOf() 和 isAttribute()。

[pure virtual] void QDesignerPropertySheetExtension::setChanged(int index, bool changed)

根据changed（更改）的参数设置给定index（索引）处的属性是否与其默认值不同。

请参考 indexOf() 和 isChanged()。

[pure virtual] void QDesignerPropertySheetExtension::setProperty(int index, const QVariant &value)

设置给定index（索引）处的属性value（值）。

警告：如果使用此函数更改属性的值，则不会更新撤消堆栈。为了确保可以使用撤消堆栈恢复属性的值，您必须使用QDesignerFormWindowCursorInterface::setProperty () 函数或其伙伴setWidgetProperty()代替。

请参考 indexOf(), property(), and propertyGroup().

[pure virtual] void QDesignerPropertySheetExtension::setPropertyGroup(int index, const QString &group)

将给定index（索引）处的属性的属性组设置为group。

将属性与组相关联会使其显示在属性编辑器中该组的部分中。任何属性的默认属性组都是定义它的类的名称。例如，QObject::objectName属性出现在QObject属性组中。

请参考indexOf ()、property ()和propertyGroup ()。

[pure virtual] void QDesignerPropertySheetExtension::setVisible(int index, bool visible)

如果visible为true，则给定index（索引）处的属性在Qt 设计师的属性编辑器中可见；否则该属性是隐藏的。

请参考 indexOf() 和 isVisible()。
