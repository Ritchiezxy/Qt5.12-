详细信息

QDesignerFormWindowInterface类允许您查询和操作出现在Qt设计师的工作空间表单窗口。

QDesignerFormWindowInterface提供有关关联表单窗口的信息，并允许更改其属性。该接口不打算直接实例化，而是提供对Qt 设计师的窗体窗口管理器控制的Qt 设计师当前窗体窗口的访问。

如果您正在寻找包含特定组件的表单窗口，您可以使用静态QDesignerFormWindowInterface::findFormWindow () 函数：


      QDesignerFormWindowInterface *formWindow;
      formWindow = QDesignerFormWindowInterface::findFormWindow(myWidget);

但此外，您可以通过Qt 设计师的窗体窗口管理器访问任何当前窗体窗口： 使用QDesignerFormEditorInterface::formWindowManager () 函数检索管理器的接口。一旦你有了这个接口，你就可以通过QDesignerFormWindowManagerInterface::formWindow () 函数访问Qt 设计师的所有当前窗体窗口。例如：


      QList<QDesignerFormWindowInterface *> forms;
      QDesignerFormWindowInterface *formWindow;

      QDesignerFormWindowManagerInterface *manager = formEditor->formWindowManager();

      for (int i = 0; i < manager->formWindowCount(); i++) {
          formWindow = manager->formWindow(i);
          forms.append(formWindow);
      }

QDesignerCustomWidgetInterface::initialize()函数的参数提供了Qt 设计师当前的QDesignerFormEditorInterface对象(上面例子中的formEditor)的指针。当实现自定义组件插件时，你必须子类化QDesignerCustomWidgetInterface类来将你的插件暴露给Qt 设计师。

拥有表单窗口后，您可以查询其属性。例如，一个普通的自定义组件插件仅由Qt 设计师在其顶层管理，即它的任何子组件都不能在Qt 设计师的工作区中调整大小。但是QDesignerFormWindowInterface为您提供了一些功能，使您能够控制组件是否应该由Qt 设计师管理：


          if (formWindow->isManaged(myWidget))
              formWindow->manageWidget(myWidget->childWidget);

关于组件管理的完整函数列表是：isManaged ()、manageWidget ()和unmanageWidget ()。还有几个相关的信号：widgetManaged ()、widgetRemoved ()、aboutToUnmanageWidget () 和widgetUnmanaged ()。

除了控制widget的管理之外，还可以使用selectWidget ()、clearSelection ()和emitSelectionChanged ()函数，以及selectionChanged ()信号来控制表单窗口中的当前选择。

您还可以使用absoluteDir()检索关于表单存储位置的信息，使用includeHints()检索包含文件，使用layoutDefault()、layoutFunction()和pixmapFunction()检索布局和pixmap函数。您可以使用isDirty()函数来确定表单窗口是否已被修改(但未保存)。您可以检索作者(),其dir(),它的fileName(),相关comment()和exportMacro(),其mainContainer(),它的features(),其grid()和resourceFiles()。

该接口为您提供了功能和插槽，允许您更改大部分信息。例外是存储表单窗口的目录。最后，有几个信号与上述信息和一般表单窗口的变化相关。

请参考QDesignerFormWindowCursorInterface、QDesignerFormEditorInterface和QDesignerFormWindowManagerInterface。

成员类型文档

enum QDesignerFormWindowInterface::FeatureFlag
flags QDesignerFormWindowInterface::Feature

此枚举描述了可用且可由表单窗口界面控制的功能。这些值在查询表单窗口以确定它支持哪些功能时使用：

约束价值价值
QDesignerFormWindowInterface::EditFeature0x01表格编辑
QDesignerFormWindowInterface::GridFeature0x02用于编辑的网格显示和对齐网格功能
QDesignerFormWindowInterface::TabOrderFeature0x04标签订单管理
QDesignerFormWindowInterface::DefaultFeatureEditFeature | GridFeature支持默认功能（表单编辑和网格）

Feature 类型是QFlags <FeatureFlag>的 typedef 。它存储 FeatureFlag 值的 OR 组合。

请参考hasFeature ()和features ()。

enum QDesignerFormWindowInterface::ResourceFileSaveMode

这个枚举描述了资源文件是如何保存的。

ConstantValue Description
QDesignerFormWindowInterface::SaveAllResourceFiles0 保存所有资源文件。
QDesignerFormWindowInterface::SaveOnlyUsedResourceFiles1 保存表单使用的资源文件。
QDesignerFormWindowInterface::DontSaveResourceFiles2 不保存资源文件。

这个枚举是在 Qt 5.0 中引入或修改的。
