详细信息

QDesignerTaskMenuExtension类允许您自定义菜单项添加到Qt设计师的任务菜单。

QDesignerTaskMenuExtension提供了一个用于创建自定义任务菜单扩展的接口。它通常用于在Qt 设计师 中创建特定于插件的任务菜单条目。

Qt 设计师使用QDesignerTaskMenuExtension来提供其任务菜单。每当请求任务菜单时，Qt 设计师将查询所选组件的任务菜单扩展。

任务菜单扩展是QActions的集合。当带有指定扩展名的插件被选中时，操作作为条目出现在任务菜单中。上面的图片显示了自定义编辑状态...在Qt Designer的默认任务菜单项之外出现的动作:剪切、复制、粘贴等。

要创建自定义任务菜单扩展，您的扩展类必须从QObject和QDesignerTaskMenuExtension继承。例如：


  class MyTaskMenuExtension : public QObject,
          public QDesignerTaskMenuExtension
  {
      Q_OBJECT
      Q_INTERFACES(QDesignerTaskMenuExtension)

  public:
      MyTaskMenuExtension(MyCustomWidget *widget, QObject *parent);

      QAction *preferredEditAction() const;
      QList<QAction *> taskActions() const;

  private slots:
      void mySlot();

  private:
      MyCustomWidget *widget;
      QAction *myAction;
  };

因为我们正在实现一个接口，所以我们必须确保使用Q_INTERFACES()宏让元对象系统知道它。这使Qt Designer能够使用qobject_cast()函数来查询所支持的接口，但只能使用QObject指针。

您必须重新实现taskActions () 函数以返回将包含在Qt 设计师任务菜单中的操作列表。或者，您可以重新实现preferredEditAction () 函数以设置在选择插件并按F2时调用的操作。首选编辑操作必须是taskActions ()返回的操作之一，如果未定义，按F2键将被忽略。

在Qt 设计师 中，除非需要扩展，否则不会创建扩展。例如，当您在Qt 设计师工作区中的组件上单击鼠标右键时，会创建一个任务菜单扩展。因此，您还必须使用QExtensionFactory或子类构造扩展工厂，并使用Qt 设计师的扩展管理器注册它。

当需要任务菜单扩展时，Qt 设计师的扩展管理器将运行所有已注册的工厂，为每个工厂调用QExtensionFactory::createExtension ()，直到找到能够为所选组件创建任务菜单扩展的工厂。然后这个工厂将创建扩展的一个实例。

Qt 设计师 中有四种可用的扩展类型：QDesignerContainerExtension、QDesignerMemberSheetExtension、QDesignerPropertySheetExtension和QDesignerTaskMenuExtension。无论请求的扩展与容器、成员表、属性表还是任务菜单相关联，Qt 设计师的行为都是相同的。

所述QExtensionFactory类提供的标准扩展工厂，并且还可以用作用于定义扩展工厂的接口。您可以创建一个新的QExtensionFactory并重新实现QExtensionFactory::createExtension () 函数。例如：


  QObject *ANewExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      if (iid != Q_TYPEID(QDesignerTaskMenuExtension))
          return 0;

      if (MyCustomWidget *widget = qobject_cast<MyCustomWidget*>(object))
          return new MyTaskMenuExtension(widget, parent);

      return 0;
  }

或者您可以使用现有工厂，扩展QExtensionFactory::createExtension () 函数，使工厂也能够创建任务菜单扩展。例如：


  QObject *AGeneralExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      MyCustomWidget *widget = qobject_cast<MyCustomWidget*>(object);

      if (widget && (iid == Q_TYPEID(QDesignerContainerExtension))) {
          return new MyContainerExtension(widget, parent);

      } else if (widget && (iid == Q_TYPEID(QDesignerTaskMenuExtension))) {
          return new MyTaskMenuExtension(widget, parent);

      } else {
          return 0;
      }
  }

有关使用QDesignerTaskMenuExtension类的完整示例，请参阅任务菜单扩展示例。该示例展示了如何为Qt 设计师创建自定义组件插件，以及如何使用QDesignerTaskMenuExtension类将自定义项添加到Qt 设计师的任务菜单。

请参考QExtensionFactory、QExtensionManager和创建自定义组件扩展。
