详细信息

该QDesignerCustomWidgetCollectionInterface类可以包含在一个单一的库中的多个自定义部件。

实现自定义组件插件时，您将其构建为单独的库。如果要在同一个库中包含多个自定义组件插件，则必须另外子类QDesignerCustomWidgetCollectionInterface。

QDesignerCustomWidgetCollectionInterface包含一个返回集合的QDesignerCustomWidgetInterface对象列表的函数。例如，如果您有多个自定义组件CustomWidgetOne,CustomWidgetTwo和CustomWidgetThree，则类定义可能如下所示：


  #include customwidgetoneinterface.h
  #include customwidgettwointerface.h
  #include customwidgetthreeinterface.h

  #include <Qt设计师/qtdesigner.h>
  #include <QtCore/qplugin.h>

  class MyCustomWidgets: public QObject, public QDesignerCustomWidgetCollectionInterface
  {
      Q_OBJECT
      Q_PLUGIN_METADATA(IID "org.qt-project.Qt.QDesignerCustomWidgetCollectionInterface")
      Q_INTERFACES(QDesignerCustomWidgetCollectionInterface)

  public:
      MyCustomWidgets(QObject *parent = 0);

      QList<QDesignerCustomWidgetInterface*> customWidgets() const override;

  private:
      QList<QDesignerCustomWidgetInterface*> widgets;
  };

在类构造函数中，您将自定义组件的接口添加到您在customWidgets () 函数中返回的列表中：


  MyCustomWidgets::MyCustomWidgets(QObject *parent)
          : QObject(parent)
  {
      widgets.append(new CustomWidgetOneInterface(this));
      widgets.append(new CustomWidgetTwoInterface(this));
      widgets.append(new CustomWidgetThreeInterface(this));
  }

  QList<QDesignerCustomWidgetInterface*> MyCustomWidgets::customWidgets() const
  {
      return widgets;
  }

请注意，不是使用Q_PLUGIN_METADATA () 宏导出每个自定义组件插件，而是导出整个集合。所述Q_PLUGIN_METADATA（）宏确保Qt设计可以访问并构建定制的窗口组件。没有这个宏，Qt 设计师 就无法使用它们。

请参考QDesignerCustomWidgetInterface和为 Qt 设计师 创建自定义组件。
