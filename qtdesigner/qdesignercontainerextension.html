详细信息

QDesignerContainerExtension类可以让你的网页添加到Qt设计师的工作空间中的自定义多页的容器。

QDesignerContainerExtension提供用于创建自定义容器扩展的接口。容器扩展由Qt 设计师管理多页面容器插件所需的一组功能和容器页面列表组成。

警告：这通常不是容器插件的扩展，只是自定义多页容器。

要创建容器扩展，您的扩展类必须从QObject和QDesignerContainerExtension继承。例如：


  class MyContainerExtension : public QObject,
         public QDesignerContainerExtension
  {
      Q_OBJECT
      Q_INTERFACES(QDesignerContainerExtension)

  public:
      MyContainerExtension(MyCustomWidget *widget,
                           QObject *parent = 0);
      int count() const;
      QWidget *widget(int index) const;
      int currentIndex() const;
      void setCurrentIndex(int index);
      void addWidget(QWidget *widget);
      void insertWidget(int index, QWidget *widget);
      void remove(int index);

  private:
      MyCustomWidget *myWidget;
  };

由于我们正在实现一个接口，我们必须确保使用Q_INTERFACES()宏让元对象系统知道它。这使Qt Designer能够使用qobject_cast()函数来查询所支持的接口，只使用QObject指针。

你必须重新实现几个功能使Qt设计器管理自定义多页面容器组件:Qt设计器使用count()来跟踪页面数量在你的容器,widget()返回的页面在给定索引容器的列表页面,并且currentIndex()返回选择页面的索引列表。Qt 设计师使用addWidget()函数将给定的页面添加到容器中，期望它被添加到页面列表中，而它期望insertWidget()函数将给定的页面插入到给定的索引中，从而将其添加到容器中。

在Qt 设计师中，只有在需要时才创建扩展。出于这个原因，您还必须创建一个QExtensionFactory，即一个能够创建扩展实例的类，并使用Qt Designer的扩展管理器注册它。

当需要容器扩展时，Qt 设计师的扩展管理器将运行所有已注册的工厂，为每个工厂调用QExtensionFactory::createExtension ()，直到找到第一个能够创建容器扩展的工厂。然后这个工厂将为插件创建扩展。

Qt 设计师 中有四种可用的扩展类型：QDesignerContainerExtension、QDesignerMemberSheetExtension、QDesignerPropertySheetExtension和QDesignerTaskMenuExtension。无论请求的扩展与多页容器、成员表、属性表还是任务菜单相关联，Qt 设计师的行为都是相同的。

QExtensionFactory类提供的标准扩展工厂，并且还可以用作用于定义扩展工厂的接口。您可以创建一个新的QExtensionFactory并重新实现QExtensionFactory::createExtension () 函数。例如：


  QObject *ANewExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      if (iid != Q_TYPEID(QDesignerContainerExtension))
          return 0;

      if (MyCustomWidget *widget = qobject_cast<MyCustomWidget*>
             (object))
          return new MyContainerExtension(widget, parent);

      return 0;
  }

或者您可以使用现有工厂，扩展QExtensionFactory::createExtension () 函数，使工厂也能够创建容器扩展。例如：


  QObject *AGeneralExtensionFactory::createExtension(QObject *object,
          const QString &iid, QObject *parent) const
  {
      MyCustomWidget *widget = qobject_cast<MyCustomWidget*>(object);

      if (widget && (iid == Q_TYPEID(QDesignerTaskMenuExtension))) {
          return new MyTaskMenuExtension(widget, parent);

      } else if (widget && (iid == Q_TYPEID(QDesignerContainerExtension))) {
          return new MyContainerExtension(widget, parent);

      } else {
          return 0;
      }
  }

有关使用QDesignerContainerExtension类的完整示例，请参阅容器扩展示例。该示例展示了如何为Qt 设计师创建自定义多页插件。

请参考QExtensionFactory、QExtensionManager和创建自定义组件扩展。

成员函数文档

[virtual] QDesignerContainerExtension::~QDesignerContainerExtension()

销毁扩展。

[pure virtual] void QDesignerContainerExtension::addWidget(QWidget *page)

通过将给定page（页面）附加到扩展的页面列表，将给定page（页面）添加到容器中。

请参考insertWidget ()、remove ()和widget ()。

[virtual] bool QDesignerContainerExtension::canAddWidget() const

返回是否可以添加组件。这确定是否启用添加或插入页面的上下文菜单选项。

对于具有单个固定页面的容器，例如QScrollArea或QDockWidget，这应该返回 false 。

这个函数是在 Qt 5.0 中引入的。

请参考addWidget ()和canRemove ()。

[virtual] bool QDesignerContainerExtension::canRemove(int index) const

返回是否可以删除给定index （索引）处的组件。这确定是否启用删除当前页面的上下文菜单选项。

对于具有单个固定页面的容器，例如QScrollArea或QDockWidget，这应该返回 false 。

这个函数是在 Qt 5.0 中引入的。

请参考remove ()和canAddWidget ()。

[pure virtual] int QDesignerContainerExtension::count() const

返回容器中的页数。

[pure virtual] int QDesignerContainerExtension::currentIndex() const

返回容器中当前选定页面的索引。

请参考 setCurrentIndex()。

[pure virtual] void QDesignerContainerExtension::insertWidget(int index, QWidget *page)

通过将给定page（页面）插入到扩展的页面列表中的给定index（索引）处，将给定page(页面)添加到容器中。

请参考addWidget ()、remove ()和widget ()。

[pure virtual] void QDesignerContainerExtension::remove(int index)

从扩展的页面列表中删除给定index(索引)处的页面。

请参考addWidget ()和insertWidget ()。

[pure virtual] void QDesignerContainerExtension::setCurrentIndex(int index)

将容器中当前选定的页面设置为扩展的页面列表中给定index(索引)处的页面。

请参考 currentIndex()。

[pure virtual] QWidget *QDesignerContainerExtension::widget(int index) const

返回扩展的页面列表中给定index（索引）处的页面。

请参考addWidget ()和insertWidget ()。
