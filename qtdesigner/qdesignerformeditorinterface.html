详细信息

该QDesignerFormEditorInterface类允许您访问的Qt 设计师的各种组件。

Qt 设计师的当前QDesignerFormEditorInterface对象保存有关Qt 设计师的所有组件的信息：动作编辑器、对象检查器、属性编辑器、组件框以及扩展和表单窗口管理器。QDesignerFormEditorInterface包含一组为所有这些组件提供接口的函数。它们通常用于查询（和操作）相应的组件。例如：


          QDesignerObjectInspectorInterface *objectInspector = 0;
          objectInspector = formEditor->objectInspector();

          QDesignerFormWindowManagerInterface *manager = 0;
          manager = formEditor->formWindowManager();

          objectInspector->setFormWindow(manager->formWindow(0));

QDesignerFormEditorInterface不打算直接实例化。QDesignerCustomWidgetInterface::initialize () 函数的参数提供了指向Qt 设计师当前QDesignerFormEditorInterface对象（formEditor在上面的示例中）的指针。实现自定义组件插件时，您必须继承 QDesignerCustomWidgetInterface以将您的插件公开给Qt 设计师。

QDesignerFormEditorInterface还提供了可以设置动作编辑器、属性编辑器、对象检查器和组件框的功能。这些仅在您想提供自己的自定义组件时有用。

如果设计师嵌入到另一个程序中，则可以提供自己的设置管理器。Qt 设计师的组件使用管理器来存储/检索持久配置设置。默认管理器使用QSettings作为后端。

最后，QDesignerFormEditorInterface提供了topLevel () 函数，该函数返回Qt 设计师的顶级组件。

请参考 QDesignerCustomWidgetInterface。
