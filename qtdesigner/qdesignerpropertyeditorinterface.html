详细信息

QDesignerPropertyEditorInterface类允许您查询和操作Qt设计师的属性编辑器的当前状态

QDesignerPropertyEditorInterface包含一组函数，这些函数通常用于查询属性编辑器的当前状态，以及几个操作其状态的插槽。该接口还提供了一个信号，propertyChanged ()，每当属性编辑器中的属性发生变化时都会发出该信号。信号的参数是更改的属性及其新值。

例如，在实现自定义组件插件时，您可以将信号连接到自定义插槽：


          QDesignerPropertyEditorInterface *propertyEditor = 0;
          propertyEditor = formEditor->propertyEditor();

          connect(propertyEditor, SIGNAL(propertyChanged(QString,QVariant)),
                  this, SLOT(checkProperty(QString,QVariant)));

然后，当属于特定组件的指定属性发生更改时，自定义槽可以检查新值是否在我们想要的范围内：


          void checkProperty(QString property, QVariant value) {
              QDesignerPropertyEditorInterface *propertyEditor = 0;
              propertyEditor = formEditor->propertyEditor();

              QObject *object = propertyeditor->object();
              MyCustomWidget *widget = qobject_cast<MyCustomWidget>(object);

              if (widget && property == aProperty && value != expectedValue)
                  {...}
          }

QDesignerPropertyEditorInterface类不打算直接实例化。你可以使用QDesignerFormEditorInterface::propertyEditor()函数来检索Qt Designer属性编辑器的接口。QDesignerCustomWidgetInterface::initialize()函数的参数提供了一个指向Qt设计师当前QDesignerFormEditorInterface对象(上面例子中的formEditor)的指针。当实现自定义组件插件时，你必须子类化QDesignerCustomWidgetInterface来将你的插件暴露给Qt Designer。

访问属性编辑器的功能是核心（）函数，您可以使用检索到表单编辑器的界面中，currentPropertyName（）函数返回的属性编辑器当前选择的属性的名称，该对象（）函数返回Qt 设计师工作区中当前选定的对象，如果属性编辑器被写保护（否则为 false），则返回 true的isReadOnly () 函数。

操作属性编辑器状态的槽是可用于更改Qt 设计师工作区中当前选定对象的setObject ()槽、更改给定属性值的setPropertyValue () 槽和用于更改给定属性值的setReadOnly () 槽控制属性编辑器的写保护。

请参考 QDesignerFormEditorInterface。

成员函数文档

QDesignerPropertyEditorInterface::QDesignerPropertyEditorInterface(QWidget *parent, Qt::WindowFlags flags = ...)

使用给定的父级和指定的窗口flags（标志）构造一个属性编辑器界面。

[virtual] QDesignerPropertyEditorInterface::~QDesignerPropertyEditorInterface()

销毁属性编辑器界面。

[virtual] QDesignerFormEditorInterface *QDesignerPropertyEditorInterface::core() const

返回一个指向Qt 设计师的当前QDesignerFormEditorInterface对象的指针。

[pure virtual] QString QDesignerPropertyEditorInterface::currentPropertyName() const

返回属性编辑器中当前选定属性的名称。

请参考 setPropertyValue()。

[pure virtual] bool QDesignerPropertyEditorInterface::isReadOnly() const

如果属性编辑器被写保护，则返回真；否则为假。

请参考 setReadOnly()。

[pure virtual] QObject *QDesignerPropertyEditorInterface::object() const

返回Qt 设计师工作区中当前选定的对象。

请参考 setObject()。

[signal] void QDesignerPropertyEditorInterface::propertyChanged(const QString &name, const QVariant &value)

每当属性编辑器中的属性发生更改时，都会发出此信号。更改的属性及其新值分别由name(名称)和value(值)指定。

请参考 setPropertyValue()。

[pure virtual slot] void QDesignerPropertyEditorInterface::setObject(QObject *object)

将Qt 设计师工作区中当前选定的对象更改为object。

请参考 object().

[pure virtual slot] void QDesignerPropertyEditorInterface::setPropertyValue(const QString &name, const QVariant &value, bool changed = ...)

将name指定的属性的值设置为value。

此外，该属性在属性编辑器中被标记为已changed （更改），即其值与默认值不同。

请参考currentPropertyName ()和propertyChanged ()。

[pure virtual slot] void QDesignerPropertyEditorInterface::setReadOnly(bool readOnly)

如果readOnly为真，则属性编辑器被写保护；否则写保护被删除。

请参考 isReadOnly()。
