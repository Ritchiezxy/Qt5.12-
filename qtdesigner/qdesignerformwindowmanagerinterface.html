详细信息

QDesignerFormWindowManagerInterface类允许您在Qt设计师中操作窗体窗口集合，并控制Qt设计师的窗体编辑操作。

QDesignerFormWindowManagerInterface不打算直接实例化。Qt 设计师使用窗体窗口管理器来控制其工作区中的各种窗体窗口。您可以使用QDesignerFormEditorInterface::formWindowManager () 函数检索Qt 设计师的窗体窗口管理器的接口。例如：


          QDesignerFormWindowManagerInterface *manager = 0;
          QDesignerFormWindowInterface *formWindow = 0;

          manager = formEditor->formWindowManager();
          formWindow = manager->formWindow(0);

          manager->setActiveFormWindow(formWindow);

在实现自定义组件插件时，QDesignerCustomWidgetInterface::initialize () 函数的参数提供了指向Qt 设计师当前QDesignerFormEditorInterface对象（formEditor在上面的示例中）的指针。您必须继承 QDesignerCustomWidgetInterface才能将您的插件公开给 Qt 设计师。

窗体窗口管理器接口提供了createFormWindow () 函数，该函数使您可以使用addFormWindow () 槽创建一个新的窗体窗口，您可以将其添加到管理器维护的窗体窗口集合中。它还提供了formWindowCount () 函数返回当前在管理器控制下的窗体窗口的数量，formWindow () 函数返回与给定索引关联的窗体窗口，以及activeFormWindow () 函数返回当前选择的窗体窗口。该removeFormWindow（）插槽可让您减少的形式窗口管理器必须保持数量和setActiveFormWindow() 插槽允许您更改Qt 设计师工作区中的表单窗口焦点。

此外，QDesignerFormWindowManagerInterface包含一组函数，使您能够干预和控制Qt 设计师的表单编辑操作。所有这些函数都返回原始动作，从而可以在干预后进一步传播该函数。

最后，该接口提供了三个信号，分别在添加表单窗口、当前选择的表单窗口更改或删除表单窗口时发出。所有信号都携带有问题的窗体窗口作为它们的参数。

请参考QDesignerFormEditorInterface和QDesignerFormWindowInterface。
