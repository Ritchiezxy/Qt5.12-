自定义 Qt 设计师表单

在Qt Designer 中保存表单时，它会存储为 UI 文件。一些表单设置，例如网格设置或默认布局的边距和间距，与表单的组件一起存储。这些设置在uic生成表单的 C++ 代码时使用。有关如何在应用程序中使用表单的更多信息，请参阅在应用程序中使用设计师 UI 文件部分。

修改表单设置

要修改表单设置，打开表单菜单并选择表单设置...

在表单设置对话框中，您可以指定表单的作者。

可以更改表单默认布局 ( Layout Default )的边距和间距属性。这些默认布局属性将被相应的Layout Function替换，如果指定了该函数，则在uic为表单生成代码时。表单设置对话框允许您指定边距和间距的功能。

Layout Function（布局函数）

当为表单生成代码时，默认布局属性将被相应的Layout Function（布局函数）替换uic。当不同的环境需要相同表单的不同布局时，这很有用。

要为表单的边距和间距指定布局功能，请选中Layout Function（布局函数）组框以启用行编辑。

您还可以指定表单的Include Hints；即，提供一个头文件列表，然后将这些头文件包含在表单窗口的关联 UI 文件中。头文件可以是本地的，即相对于项目目录"mywidget.h"，或全局的，即 Qt 或编译器标准库的一部分：<QtWidgets/QWidget>.

最后，您可以指定用于将像素图加载到表单窗口中的函数（像素图函数）。
