成员函数文档

[override virtual] QWidget *WidgetFactory::containerOfWidget(QWidget *w) const

当w是设计器已知的容器时，返回应将子项插入其中的组件。

通常是w本身，但也有例外（例如，一个tabwidget被设计者称为容器，但是子widget应该插入到tabwidget的当前页面中。在这种情况下，tabwidget的当前页面会被返回。）

[override virtual] QLayout *WidgetFactory::createLayout(QWidget *widget, QLayout *parentLayout, int type) const

在组件widget 创建类型为type（HBox, VBox 或 Grid） 的布局器。

[override virtual] QWidget *WidgetFactory::widgetOfContainer(QWidget *w) const

返回的实际设计师组件容器w。这通常是w本身,但它可能是一个w的父级别或比父级更高的级别(例如,当处理一个tabwidget并且w是一个容器并且包含和布局了子组件，但设计师所知道的实际组件是w的父组件tabwidget。在这种情况下，将返回tabwidget。)
