QFormBuilder

用于在运行时从 UI 文件动态构建用户界面

详细信息

此外，QFormBuilder类提供了在运行时从 UI 文件构建用户界面的可能性。

要包含模块类的定义，请使用以下指令：


  #include <Qt设计师>

要链接模块，请将此行添加到您的qmake.pro 文件中：


  CONFIG += designer
