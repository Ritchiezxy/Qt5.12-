使用Qt设计师编辑资源

Qt Designer完全支持Qt资源系统，使资源可以在设计时与表单一起指定。为了帮助设计人员和开发人员管理应用程序的资源，Qt Designer的资源编辑器允许按表单定义资源。换句话说，每个表单都可以有一个单独的资源文件。

定义资源文件

要指定资源文件，必须通过打开“Tools ”菜单并切换“Resource Browser”选项来启用资源编辑器。

资源文件

在资源浏览器中，您可以打开现有的资源文件或创建新的资源文件。点击 Edit Resources 按钮 来编辑您的资源。要重新加载资源，请单击 Reload 按钮 。

一旦资源文件被加载，你可以使用给定的 Add Files 和Remove Files 按钮, 创建或删除条目，并使用 Add Files 按钮 指定资源(例如，图像)。注意，这些资源必须驻留在当前资源文件的目录或它的一个子目录中。

编辑资源文件

按下 按钮向文件中添加一个新的资源条目。然后使用 Add Files 按钮 来指定资源。

您可以通过在资源编辑器中选择相应的条目并按 按钮来删除资源。

使用的资源

一旦定义了资源，您就可以在编写表单时主动使用它们。例如，您可能想要使用资源文件中指定的图标创建一个工具按钮。

使用资源

当使用可能在资源文件中定义的值更改属性时，Qt Designer的属性编辑器允许您指定一个资源，除了以普通方式选择源文件的选项之外。

选择一个资源

您可以通过单击Choose Resource...来打开资源选择器。在设计过程中随时添加资源。
