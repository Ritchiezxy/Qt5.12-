All QML Modules

These are the QML API pages for Qt 5's QML modules.

Qt.labs.folderlistmodel

FolderListModel提供文件系统文件夹内容的模型

Qt.labs.location

提供实验性QtLocation QML类型，如Navigator(导航器）和各种映射对象类型(不要与映射项混淆)

Qt.labs.qmlmodels

为数据模型提供QML实验类型

Qt.labs.settings

提供与平台无关的持久应用程序设置

Qt.labs.wavefrontmesh

WavefrontMesh提供了一个基于Wavefront .obj file文件的网格

QtQml.Models

为数据模型提供QML类型

QtQuick.Dialogs

为标准文件、颜色选择器和消息对话框提供QML类型

QtQuick.Shapes

提供用于绘制描边和填充形状的QML类型

QtQuick.VirtualKeyboard.Settings

为Qt Virtual Keyboard（Qt虚拟键盘）提供设置

QtQuick.VirtualKeyboard.Styles

为Qt Virtual Keyboard（Qt虚拟键盘）提供样式

QtQuick.Window

为窗口管理提供QML类型

QtQuick.XmlListModel

提供QML类型，用于从XML数据创建模型

Qt3D.Core

提供核心Qt 3D QML类型

Qt3D.Animation

为动画模块提供Qt 3D QML类型

Qt3D.Extras

为extras模块提供Qt 3D QML类型

Qt3D.Input

为Qt 3D用户输入提供QML类型

Qt3D.Logic

提供QML类型，以便与3D后端同步帧

Qt3D.Render

提供Qt 3D QML类型用于渲染

QtQuick.Scene2D

为scene2d模块提供Qt 3D QML类型

QtQuick.Scene3D

为Scene3D模块提供Qt 3D QML类型

QtCharts

QQt Charts API的QML类型

QtBluetooth

为设备上的基本蓝牙操作提供QML类型

QtNfc

提供用于访问NFC Forum Tags的QML类型

QtDataVisualization

Qt Data Visualization API（Qt数据可视化API）的QML类型

QtQml

Qt QML模块提供的QML类型列表

QtQml.StateMachine

提供QML类型来创建和执行状态图

QtQuick.Particles

为粒子效果提供QML类型

QtQuick.Layouts

提供QML类型，用于在用户界面中排列QML项

QtQuick

提供图形化QML类型

QtTest

此模块提供QML类型来对QML应用程序进行单元测试

QtGamepad

为gamepad硬件提供的QML类型

QtGraphicalEffects

Qt图形效果模块提供了一系列惊人的视觉效果，可以应用于Qt Quick应用程序

QtLocation

为映射和位置信息提供QML类型

QtPositioning

提供位置信息的QML类型

QtAudioEngine

为3D位置音频播放和内容管理提供QML类型

QtMultimedia

为多媒体支持提供QML类型

QtPurchasing

为应用程序内购买提供QML类型

QtQuick.Controls

为用户界面提供QML类型

QtQuick.Controls.Styles

为Qt Quick控件样式提供QML类型

QtQuick.Extras

为用户界面提供QML类型

QtQuick.Controls

为用户界面提供QML类型(Qt Quick Controls 2)

QtRemoteObjects

为远程对象支持提供QML类型

QtScxml

使用QML启用SCXML状态机

QtSensors

提供读取传感器数据的类型

QtQuick.VirtualKeyboard

为输入框架和引用键盘前端提供QML类型

QtWayland.Compositor

QtWebChannel

提供WebChannel功能的QML类型列表

QtWebView

QtWinExtras
