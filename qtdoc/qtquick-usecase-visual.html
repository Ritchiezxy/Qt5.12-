用例——QML中的可视化元素

Rectangle类型

对于最基本的视觉效果， Qt Quick 提供了一个Rectangle 类型来绘制矩形。这些矩形可以用颜色或垂直渐进着色Rectangle 类型还可以在矩形上绘制边框。

若要绘制矩形之外的自定义形状，请参阅Canvas 类型或使用 Image 显示预呈现的图像。


  import QtQuick 2.3

  Item {

      width: 320
      height: 480

      Rectangle {
          color: "#272822"
          width: 320
          height: 480
      }

      // This element displays a rectangle with a gradient and a border
      Rectangle {
          x: 160
          y: 20
          width: 100
          height: 100
          radius: 8 // This gives rounded corners to the Rectangle
          gradient: Gradient { // This sets a vertical gradient fill
              GradientStop { position: 0.0; color: "aqua" }
              GradientStop { position: 1.0; color: "teal" }
          }
          border { width: 3; color: "white" } // This sets a 3px wide black border to be drawn
      }

      // This rectangle is a plain color with no border
      Rectangle {
          x: 40
          y: 20
          width: 100
          height: 100
          color: "red"
      }
  }

Image类型

Qt Quick 提供了一种可以用来显示图片的Image 类型。Image 类型有一个source（源）属性，其值可以是远程或本地URL，也可以是嵌入在已编译资源文件中的图片文件的URL。


  // This element displays an image. Because the source is online, it may take some time to fetch
  Image {
      x: 40
      y: 20
      width: 61
      height: 73
      source: "http://codereview.qt-project.org/static/logo_qt.png"
  }

对于更复杂的图像，还有其他类型类似于Image. BorderImage 绘制具有网格缩放的图像，适用于用作边框的图像。 AnimatedImage 可以播放gif和mng动画。 AnimatedSprite 和 SpriteSequence 以非动画格式播放由相邻存储的多个帧组成的动画。

要显示视频文件和摄像机数据，请参阅Qt多媒体 模块。

共享视觉属性

Qt Quick提供的所有可视项都基于item类型，它为可视项提供了一组常见的属性，包括不透明度和转换属性。

不透明度和可见性

Qt Quick提供的QML对象类型内置了对 opacity（不透明）的支持。 不透明度可以被动画化，以允许平滑地过渡到透明状态或从透明状态过渡到透明状态。可视性还可以使用visible（可视） 属性更有效地进行管理，但代价是不能使其具有动画效果。


  import QtQuick 2.3

  Item {

      width: 320
      height: 480

      Rectangle {
          color: "#272822"
          width: 320
          height: 480
      }

      Item {
          x: 20
          y: 270
          width: 200
          height: 200
          MouseArea {
              anchors.fill: parent
              onClicked: topRect.visible = !topRect.visible
          }
          Rectangle {
              x: 20
              y: 20
              width: 100
              height: 100
              color: "red"
          }
          Rectangle {
              id: topRect
              opacity: 0.5

              x: 100
              y: 100
              width: 100
              height: 100
              color: "blue"
          }
      }
  }

转换

Qt Quick类型内置了对转换的支持。如果希望旋转或缩放可视内容，可以设置Item::rotation 或 Item::scale 属性。这些也可以是动画的。


  import QtQuick 2.3

  Item {

      width: 320
      height: 480

      Rectangle {
          color: "#272822"
          width: 320
          height: 480
      }

      Rectangle {
          rotation: 45 // This rotates the Rectangle by 45 degrees
          x: 20
          y: 160
          width: 100
          height: 100
          color: "blue"
      }
      Rectangle {
          scale: 0.8 // This scales the Rectangle down to 80% size
          x: 160
          y: 160
          width: 100
          height: 100
          color: "green"
      }
  }

有关更复杂的转换，请参见Item::transform属性。
