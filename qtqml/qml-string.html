string QML基本类型

一个自由格式的文本字符串。

string 类型指的是带引号的自由格式文本字符串，例如:“Hello world !”

示例：


  Text { text: "Hello world!" }

字符串有一个length 属性，用于保存字符串中的字符数量。

QML使用arg()函数扩展了JavaScript String类型，以支持值替换。

当与C++集成时，请注意任何从C++传递到QML的QString值都会自动转换为string 值，反之亦然。

这个基本类型由QML语言提供。

请参考 QML基本类型。
