point QML基本类型

具有x和y属性的值。

point 类型指的是具有x和y 属性的值。

要创建一个point 值，指定它为“x,y”字符串:


  CustomObject { myPointProperty: "0,20" }

或者使用Qt.point()函数:


  CustomObject { myPointProperty: Qt.point(0, 20) }

当与C++集成时，请注意从C++传递到QML的任何QPoint或QPointF值都会自动转换为一个点值。当一个点值传递给C++时，它会自动转换为QPointF值。

请参考 QML基本类型。
