real QML基本类型

有小数点的数字。

real 类型指的是带有小数点的数字，例如1.2或-29.8。

示例：


  Item { width: 100.45; height: 150.82 }

注意:在QML中，所有实数都以双精度IEEE浮点格式存储。

这个基本类型由QML语言提供。

请参考 QML基本类型.
