DelegateChoice QML 类型

封装委托以及何时使用它。 更多...

导入语句: import Qt.labs.qmlmodels 1.0

属性

  • column : index
  • delegate : Component
  • index : index
  • roleValue : string
  • row : index

详细信息

DelegateChoice类型包装了一个委托，并定义了应该选择它的环境。

DelegateChoice可以嵌套在DelegateChooser中。

请参考 DelegateChooser。

属性的文档

column : index

此属性保存用于匹配模型元素列值的值。


delegate : Component

如果该选择与模型项匹配，则此属性保存要使用的委托。


index : index

此属性保存用于匹配模型元素索引值的值。这实际上是row的别名。

请参考 row.


roleValue : string

此属性保存用于匹配由DelegateChooser::role提供的角色数据的值。


row : index

此属性保存用于匹配模型元素的行值的值。对于只有index属性(因此只有一个列)的模型，应该将此属性作为索引，并设置为所需的索引值。

注意： 设置行和索引具有未定义的行为。两者是等价的，只能使用一个。

请参考 index。
