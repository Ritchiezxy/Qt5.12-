QAbstractState QML 类型

QAbstractState类型是StateMachine状态的基本类型。 更多...

导入语句: import QtQml.StateMachine 1.12
从： Qt 5.4
继承自：

FinalState, HistoryState, 和 State

属性

  • active : bool

信号

  • entered()
  • exited()

详细信息

不要直接使用QAbstractState，而是使用State、FinalState或StateMachine代替。

请参考 StateMachine 和 State。

属性的文档

[read-only] active : bool

这种状态的活跃性质。在entered()和exited() 信号之间的状态是活跃的。此属性是只读的。

请参考 entered 和 exited。


信号文档

entered()

当状态变为活动状态时发出此信号。

对应的处理程序是onEntered。

请参考 active 和 exited。


exited()

当状态变为非活动状态时发出此信号。

相应的处理程序是onExited。

请参考 active 和 entered。
