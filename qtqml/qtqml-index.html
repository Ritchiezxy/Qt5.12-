Qt QML

Qt QML模块为使用QML语言开发应用程序和库提供了一个框架。它定义并实现了语言和引擎基础设施，并提供了一个API，使应用程序开发人员能够使用定制类型扩展QML语言，并将QML代码与JavaScript和C++集成在一起。Qt QML模块同时提供QML API 和 C++ API。

请注意，Qt QML模块为QML应用程序提供了语言和基础设施，Qt Quick模块为构建用户界面提供了许多可视化组件、模型-视图支持、动画框架等。

有关QML和Qt Quick的新特性，请参阅QML应用程序，了解编写QML应用程序的介绍。

开始

要包含模块类的定义，请使用以下指令:


  #include <QtQml>

Qt QML中的QML类型可以通过QtQml导入获得。要使用这些类型，请将以下import语句添加到.qml文件中:


  import QtQml 2.12

要链接到模块，请将这一行添加到qmake .pro文件中:


  QT += qml

QML和QML类型

Qt QML模块包含QML框架和应用程序中使用的重要QML类型。QML的构造在 QML引用中进行了描述。

除了QML基本类型外，模块还提供了以下QML对象类型:

  • Component（组件）
  • QtObject（Qt 对象）
  • Binding（绑定）
  • Connections（连接）
  • Timer（计时器）

Qt全局对象为各种QML类型提供了有用的枚举和函数。

列表和模型

新的Qt 5.1中，模型类型被移动到子模块QtQml.Models。 有关于此，在Qt QML Models页面有更多信息。
