QML的HistoryState类型

HistoryState 型提供了返回到先前活动子状态的方法。更多…

导入语句: import QtQml.StateMachine 1.12
Since: Qt 5.4
Inherits:

QAbstractState

  • 所有成员的列表，包括继承的成员

属性

  • defaultState : QAbstractState
  • historyType : enumeration

详细信息

历史状态是一个伪状态，表示父状态上次退出父状态时所在的子状态。以历史状态为目标的转换实际上是到父状态的其他子状态之一的转换。HistoryState是声明性状态机框架的一部分。

使用defaultState 属性设置如果从未输入父状态，则应该输入的状态。

示例用法


  import QtQuick 2.0
  import QtQml.StateMachine 1.0 as DSM

  Rectangle {
      Button {
          anchors.fill: parent
          id: button
          text: "Press me"
          DSM.StateMachine {
              id: stateMachine
              initialState: parentState
              running: true
              DSM.State {
                  id: parentState
                  initialState: child2
                  onEntered: console.log("parentState entered")
                  onExited: console.log("parentState exited")
                  DSM.State {
                      id: child1
                      onEntered: console.log("child1 entered")
                      onExited: console.log("child1 exited")
                  }
                  DSM.State {
                      id: child2
                      onEntered: console.log("child2 entered")
                      onExited: console.log("child2 exited")
                  }
                  DSM.HistoryState {
                      id: historyState
                      defaultState: child1
                  }
                  DSM.SignalTransition {
                      targetState: historyState

                      // Clicking the button will cause the state machine to enter the child state
                      // that parentState was in the last time parentState was exited, or the history state's default
                      // state if parentState has never been entered.
                      signal: button.clicked
                  }
              }
          }
      }
  }

默认情况下，历史状态是浅的，这意味着它不会记住嵌套的状态。这可以通过historyType属性配置。

参见StateMachine 和 State。

属性文档

defaultState : QAbstractState

此历史状态的默认状态。

默认状态指示要转换到的状态(如果以前从未输入父状态)。


historyType : enumeration

此历史状态记录的历史类型。

该属性的默认值是HistoryState.ShallowHistory。

此枚举指定HistoryState记录的历史类型。

  • HistoryState.ShallowHistory 只记录父状态的直接子状态。在本例中，以history状态为目标的转换将以父状态上次退出时的直接子状态结束。这是默认值。
  • HistoryState.DeepHistory 将记录嵌套状态。在这种情况下，以历史状态为目标的转换将最终处于父类上次退出时所在的嵌套最深的后代状态。
