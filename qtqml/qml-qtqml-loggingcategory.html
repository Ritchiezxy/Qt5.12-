LoggingCategory QML 类型

Defines a logging category in QML. 更多...

导入语句： import QtQml 2.12
从： Qt 5.8

属性

  • defaultLogLevel : enumeration
  • name : string

详细信息

日志类别可以作为第一个参数传递给console.log()和friends。如果提供给日志记录器，LoggingCategory的名称将被用作日志类别，否则将使用默认的日志类别。


  import QtQuick 2.8

  Item {
      LoggingCategory {
          id: category
          name: "com.qt.category"
          defaultLogLevel: LoggingCategory.Warning
      }

      Component.onCompleted: {
        console.log(category, "message");
      }
  }

注意： 由于创建对象的开销很大，所以建议将所需的LoggingCategory定义放到一个单例中，并在需要的地方导入它。

请参考 QLoggingCategory。

属性的文档

defaultLogLevel : enumeration

保存日志类别的默认日志级别。默认情况下，它是用LoggingCategory创建的。调试日志级别。

注意： 该属性需要在声明LoggingCategory时设置，以后不能更改。

这个属性在Qt 5.12中被引入。


name : string

保存日志类别的名称。

注意： 该属性需要在声明LoggingCategory时设置，以后不能更改。

请参考 QLoggingCategory::categoryName()。
