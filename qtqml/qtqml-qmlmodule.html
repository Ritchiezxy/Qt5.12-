Qt QML QML 类型

Qt QML模块提供了各种可与QML语言一起使用的方便类型的定义和实现，包括一些基本的QML类型，这些类型可以为进一步扩展QML语言提供基础。QtObject和Component对象类型是非可视的，并为QML的扩展提供构建块。

导入 QtQml

QtQml模块提供的类型只有在QML文档导入QtQml 名称空间时才可用(或者如果文档导入QtQuick 名称空间，如下所述)。

QtQml模块的当前版本是2.12版本，因此可以通过以下语句导入:


  import QtQml 2.12

大多数客户端永远不需要使用QtQml导入，因为所有的类型都是由QtQuick命名空间提供的，可以按如下方式导入:


  import QtQuick 2.12

有关Qt Quick名称空间及其为QML应用程序开发人员提供的信息，请参阅Qt Quick模块文档。

用于创建列表和模型的QML类型，如ListModel和ListElement，被移动到子模块QtQml.Models。Qt QML模型页面有更多信息。

以下类型的文档同样适用于Qt Quick模块提供的同名类型，因为它们实际上是相同的。

基本类型

提供了以下QML基本类型:

date

Date 值

point

带有x和y属性的值

rect

具有x, y，宽度和高度属性的值

size

具有宽度和高度属性的值

对象类型

提供了以下QML对象类型:

Binding

支持任意创建属性绑定

Component

封装一个QML组件定义

Connections

描述与信号的一般连接

Date

提供日期功能

Instantiator

动态创建对象

Locale

提供特定于区域设置的属性和格式化数据

LoggingCategory

在QML中定义日志类别

Number

对象提供表示一个数值

Qt

提供了一个全局对象和有用的枚举和函数从Qt

QtObject

基本的QML类型

String

对象表示一个字符串值

Timer

按指定的间隔触发处理程序
