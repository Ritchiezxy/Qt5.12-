方法示例在BirthdayParty 类中有一个额外的方法:invite()。invite()是用Q_INVOKABLE声明的，这样就可以从QML调用它。


      Q_INVOKABLE void invite(const QString &name);

在example.qml中，invite()方法在Component.onCompleted信号处理程序中被调用:


  import QtQuick 2.0
  import People 1.0

  BirthdayParty {
      host: Person {
          name: "Bob Jones"
          shoeSize: 12
      }
      guests: [
          Person { name: "Leo Hodges" },
          Person { name: "Jack Smith" },
          Person { name: "Anne Brown" }
      ]

      Component.onCompleted: invite("William Green")
  }
