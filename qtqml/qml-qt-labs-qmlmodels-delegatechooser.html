DelegateChooser QML 类型

允许视图对模型中不同类型的项使用不同的委托。 更多...

导入语句： import Qt.labs.qmlmodels 1.0
  • 所有成员的列表，包括继承成员

属性

  • choices : list<DelegateChoice>
  • role : string

详细信息

DelegateChooser是一种特殊的Component类型，用于视图需要组件并将其用作委托的场景。DelegateChooser封装了一组DelegateChoice。使用这些选项确定将为模型中的每个项实例化的委托。选择的执行是基于模型项的role，也基于索引。

注意： 这种类型旨在透明地只与TableView和任何基于DelegateModel的视图一起工作。在内部不基于DelegateModel的视图(包括用户定义的视图)需要显式支持这种类型的组件，以使其发挥所描述的功能。

请参考 DelegateChoice。

属性的文档

[default] choices : list<DelegateChoice>

选择器的DelegateChoices列表。

该列表被视为有序列表，其中匹配的第一个DelegateChoice将被用作一个视图。

通常不需要引用choices 属性，因为它是DelegateChooser的默认属性，因此所有子项都会自动分配给这个属性。


role : string

此属性包含用于确定给定模型项的委托的角色。

请参考 DelegateChoice。
