详细信息

QQmlScriptString类封装了一个脚本及其上下文。

QQmlScriptString 用于创建接受来自QML的脚本"assignment（赋值）"的QObject QObject属性。

通常，下列QML将导致为script属性建立绑定;也就是说，script将被分配从运行myObj.value = Math.max(myValue, 100)中获得的值 。 


  MyType {
      script: myObj.value = Math.max(myValue, 100)
  }

如果该属性具有 QQmlScriptString类型，则脚本本身——myObj.value = Math.max(myValue, 100) ——将传递给script 属性，类可以选择如何处理它。通常，类将在稍后的某个时候使用 QQmlExpression计算脚本。


  QQmlExpression expr(scriptString);
  expr.evaluate();

参见QQmlExpression。
