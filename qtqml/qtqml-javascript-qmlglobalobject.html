QML全局对象

QML JavaScript宿主环境实现了以下宿主对象和函数。这些都是内置的，可以从QML中加载的任何JavaScript代码中使用，而不需要额外的导入:

  • Qt对象:该对象特定于QML，并提供特定于QML环境的辅助方法和属性。
  • qsTr(), qsTranslate(), qsTrId(), QT_TR_NOOP(), QT_TRANSLATE_NOOP(), and QT_TRID_NOOP() functions: These functions are specific to QML, and provide translation capabilities to the QML environment.qsTr()、qsTranslate()、qsTrId()、QT_TR_NOOP()、QT_TRANSLATE_NOOP()和QT_TRID_NOOP()函数:这些函数特定于QML，并向QML环境提供翻译功能。
  • gc()函数:该函数特定于QML，并提供一种手动触发垃圾收集的方法。
  • print()函数:该函数特定于QML，并提供一种向控制台输出信息的简单方法。
  • 控制台对象:该对象实现了FireBug控制台API的一个子集。
  • XMLHttpRequest, DOMException:这些对象实现了W3C XMLHttpRequest规范的一个子集。

注意：For more information about this, see JavaScript Environment Restrictions.globalObject()函数不能用于修改QQmlEngine的全局对象。有关这方面的更多信息，请参见JavaScript环境限制。
