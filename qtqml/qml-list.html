list QML基本类型

QML对象列表。

 list 类型指的是一个QML对象列表。

列表值可以通过类似JavaScript数组的方式访问:

  • 值使用方括号语法赋值，并用逗号分隔
  • length 属性提供列表中项目的数量
  • 使用[index]语法访问列表中的值

可以使用push方法将值动态添加到列表中，就像它是一个JavaScript数组一样

list 只能存储QML对象，不能包含任何基本类型值。(要在列表中存储基本类型，请使用var类型。)

当与C++集成时，请注意任何从C++传递到QML的QQmlListProperty值都会自动转换为列表值，反之亦然。

使用列表类型

例如，Item类型有一状态s列表类型属性，可以按如下方式赋值和使用:


  import QtQuick 2.0

  Item {
      width: 100; height: 100

      states: [
          State { name: "activated" },
          State { name: "deactivated" }
      ]

      Component.onCompleted: {
          console.log("Name of first state:", states[0].name)
          for (var i = 0; i < states.length; i++)
              console.log("state", i, states[i].name)
      }
  }

已定义的State对象将按定义的顺序添加到状态列表中。

如果列表只包含一个对象，方括号可以省略:


  import QtQuick 2.0

  Item {
      width: 100; height: 100
      states: State { name: "activated" }
  }

注意，对象一旦创建，就不能单独添加或从列表中删除;要修改列表的内容，必须将其重新分配给一个新列表。

注意: 不建议将list 类型作为自定义属性的类型。因此应该使用var 类型，因为由var 类型存储的列表可以在QML中以更大的灵活性进行操作。

这个基本类型由QML语言提供。
