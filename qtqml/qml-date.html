date QML基本类型

一个日期值。

date 类型指的是一个日期值，包括一天中的时间。

要创建日期值，请将其指定为“YYYY-MM-DDThh:mm:ss.zzz”字符串。(T是文字，YYYY是完整的年份，MM和DD是月和日，hh、MM和ss是小时、分和秒，.zzz是毫秒，Z是时区偏移量。T和以下时间是可选的。如果省略，该日期将被处理为UTC日的开始，它落在某些时区的其他日期上。当包含T时，:ss。zzz或只是.zzz部分可以省略。有或没有这些，区域偏移可以省略，在这种情况下使用本地时间。)例如:


  MyDatePicker { minDate: "2000-01-01 0:0"; maxDate: "2020-12-31 23:59" }

要读取从C++扩展类返回的日期值，请使用Qt.formatDate()和Qt.formatDateTime()。

当与C++集成时，请注意任何从C++传递到QML的QDate或QDateTime值都会自动转换为日期值，反之亦然。但是请注意，转换QDate将导致UTC的一天开始，它在其他一些时区的不同日期。通过QDateTime显式地转换QDate，指定本地时间或相关时区，并选择可靠存在的一天中的某个时间(比如中午)(夏令时转换跳过一个小时，在一天的一端或另一端)，通常会更加健壮。

这个基本类型由QML语言提供。它可以隐式转换为Date对象。

请参考 QML Date object 和 QML基本类型。
