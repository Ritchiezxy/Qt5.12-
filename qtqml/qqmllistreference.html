详细信息

QQmlListReference类允许对QQmlListProperty属性进行操作。

QQmlListReference允许C++程序以简单和类型安全的方式读取QML列表属性，并为其赋值。QQmlListReference可以通过传递对象和属性名或通过QQmlProperty实例创建。这两个是等价的:


  QQmlListReference ref1(object, "children");

  QQmlProperty ref2(object, "children");
  QQmlListReference ref2 = qvariant_cast<QQmlListReference>(ref2.read());

不是所有的QML列表属性都支持所有的操作。canAppend()、canAt()、canClear()和canCount()等一组方法允许程序查询给定属性是否支持某个操作。

QML列表属性是类型安全的。只有派生自正确基类的QObject才能被赋值给列表。listElementType()方法可用于查询所支持的QObject类型的QMetaObject。试图将不正确类型的对象添加到列表属性将会失败。

与普通列表一样，当通过索引访问列表元素时，调用者有责任确保在调用at()之前不会使用count()方法请求超出范围的元素。

成员函数文档

QQmlListReference::QQmlListReference()

构造无效的实例。

QQmlListReference::QQmlListReference(QObject *object, const char *property, QQmlEngine *engine = nullptr)

为object（对象）的property（属性）构造一个QQmlListReference。如果属性不是列表属性，则创建一个无效的QQmlListReference。如果对象在引用构造之后被销毁，它将自动失效。也就是说，即使在对象被删除之后，也可以安全地保存QQmlListReference实例。

访问一些QML创建的列表属性需要传递engine （引擎）。如果有疑问，引擎是可用的，通过它。

bool QQmlListReference::append(QObject *object) const

将object （对象）追加到列表。如果操作成功则返回true，否则返回false。

请参考 canAppend()。

QObject *QQmlListReference::at(int index) const

返回index处的列表元素，如果操作失败则返回0。

请参考 canAt()。

bool QQmlListReference::canAppend() const

如果列表属性可以被附加到，则返回true，否则返回false。如果引用无效则返回false。

请参考 append()。

bool QQmlListReference::canAt() const

如果列表属性可以按索引查询，则返回true，否则返回false。如果引用无效则返回false。

请参考 at()。

bool QQmlListReference::canClear() const

如果列表属性可以被清除，则返回true，否则返回false。如果引用无效则返回false。

请参考 clear()。

bool QQmlListReference::canCount() const

如果可以查询list属性的元素计数，则返回true，否则返回false。如果引用无效则返回false。

请参考 count()。

bool QQmlListReference::clear() const

清空列表。如果操作成功则返回true，否则返回false。

请参考 canClear()。

int QQmlListReference::count() const

返回列表中对象的数量，如果操作失败则返回0。

bool QQmlListReference::isManipulable() const

如果实现了at()， count()， append()和clear()则返回true，以便您可以操作列表。

请参考 isReadable()， at()， count()， append()， 和 clear()。

bool QQmlListReference::isReadable() const

如果实现了at()和count()则返回true，以便您可以访问元素。

请参考 isManipulable()， at()， 和 count()。

bool QQmlListReference::isValid() const

如果实例引用有效的列表属性，则返回true，否则返回false。

const QMetaObject *QQmlListReference::listElementType() const

返回list属性中存储的元素的QMetaObject，如果引用无效则返回nullptr 。

可以提前使用QMetaObject来确定是否可以将给定的实例添加到列表中。

QObject *QQmlListReference::object() const

返回列表属性的对象。如果引用无效，则返回nullptr。
