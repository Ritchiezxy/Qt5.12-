属性的文档

[read-only] currentIndex : QModelIndex

使用setCurrentIndex()设置其值。

请参考 setCurrentIndex() 和 currentChanged()。


[read-only] hasSelection : bool

每次发出selectionChanged()时，它都会触发属性绑定更新，即使它的值没有改变。

请参考 selection、 selectedIndexes、 select() 和 selectionChanged()。


model : QAbstractItemModel

该属性的值必须与视图的模型匹配。


[read-only] selectedIndexes : QModelIndexList

包含选择模型中所有索引的列表。


[read-only] selection : object

保存存储在选择模型中的选择范围。


信号文档

currentChanged(QModelIndex current, QModelIndex previous)

每当当前项目发生变化时都会发出此信号。在previous的模型项指标被替换current指数作为选择的当前项。

请注意，重置项目模型时不会发出此信号。

请参考 currentIndex、setCurrentIndex() 和 selectionChanged()。


selectionChanged(QItemSelection selected, QItemSelection deselected)

当选择发生变化时，就会发出这个信号。选择中的更改表示为未选中项的项选择和选中项的项选择。

请注意，当前索引的变化与选择无关。另请注意，重置项目模型时不会发出此信号。

请参考 select() 和 currentChanged()。


方法文档

void clear()

清除选择模型。发出selectionChanged()和currentChanged()。


void clearCurrentIndex()

清除当前索引。发出currentChanged()。


void clearSelection()

清除选择模型中的选择。发出selectionChanged()。


bool columnIntersectsSelection(int column, QModelIndex parent)

如果在具有给定parent元素的column 中选择了任何项，则返回true 。


bool isColumnSelected(int column, QModelIndex parent)

如果parent索引中的中column 都被选中，则返回true 。

请注意，此函数通常比对同一列中的所有项目调用isSelected()更快，并且忽略不可选择的项目。


bool isRowSelected(int row, QModelIndex parent)

如果parent索引中的中row 都被选中，则返回true 。

请注意，此函数通常比对同一行中的所有项目调用isSelected()更快，并且忽略不可选择的项目。


bool isSelected(QModelIndex index)

如果选择了给定的模型项index ，则返回true 。


void reset()

清除选择模型。不发出任何信号。


bool rowIntersectsSelection(int row, QModelIndex parent)

如果在具有给定父对象的row 中选择了任何项，则返回true 。


void select(QModelIndex index, SelectionFlags command)

使用指定的command选择模型项index ，并发出selectionChanged()()。

命令参数的有效值为：

常量描述
NoUpdate不会进行任何选择。
Clear完整的选择将被清除。
Select将选择所有指定的索引。
Deselect将取消选择所有指定的索引。
Toggle将根据其当前状态选择或取消选择所有指定的索引。
Current当前选择将被更新。
Rows所有索引都将扩展到跨行。
Columns所有索引都将扩展到跨列。
SelectCurrent 为方便起见，提供了 Select 和 Current 的组合。
ToggleCurrentToggle 和 Current 的组合，提供方便。
ClearAndSelect为方便起见，提供了 Clear 和 Select 的组合。

void select(QItemSelection selection, SelectionFlags command)

使用指定的command选择项selection ，并发出selectionChanged()()。

select( index , command )中描述了有效的command值。


QModelIndexList selectedColumns(int row)

返回给定row中所有行都被选中的列的索引。

请参考 selectedRows()。


QModelIndexList selectedRows(int column)

返回给定column中所有列都被选中的行的索引。

请参考 selectedColumns()。


void setCurrentIndex(QModelIndex index, SelectionFlags command)

将模型项index 设置为当前项，并发出currentChanged()。当前项用于键盘导航和焦点指示；它独立于任何选定的项目，尽管选定的项目也可以是当前项目。

根据指定的command，index 也可以成为当前选择的一部分。

select( index , command )中描述了有效的命令值。

请参考 select()。
