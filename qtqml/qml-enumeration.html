enumeration QML基本类型

命名枚举值。

enumeration 类型引用命名枚举值。

每个指定的值都可以被称为<Type>.<value>。例如，Text类型有一个AlignRight 枚举值


  Text { horizontalAlignment: Text.AlignRight }

(为了向后兼容，枚举值也可以指定为字符串，例如。“AlignRight”。此表单不推荐用于新代码。)

在与C++集成时，请注意，从C++传递到QML的任何枚举值都会自动转换为枚举值，反之亦然。

这个基本类型由QML语言提供。QtQuick导入提供了一些枚举值。

使用QML中的枚举类型

enumeration 类型是C++ enum 类型的一种表示。不能在QML本身中引用enumeration 类型;相反，当从QML代码引用enumeration 值时，可以使用int或var类型。

例如:


  import QtQuick 2.0

  Item {
      // refer to Text.AlignRight using an int type
      property int enumValue: textItem.horizontalAlignment

      signal valueEmitted(int someValue)

      Text {
          id: textItem
          horizontalAlignment: Text.AlignRight
      }

      // emit valueEmitted() signal, which expects an int, with Text.AlignRight
      Component.onCompleted: valueEmitted(Text.AlignRight)
  }
