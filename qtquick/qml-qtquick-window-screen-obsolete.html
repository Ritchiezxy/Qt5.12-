Screen 的废弃成员

以下的 Screen 成员已经废弃。 提供它们是为了让旧的源代码继续工作。我们强烈建议不要在新代码中使用它们。

Attached Properties

  • logicalPixelDensity : real （废弃）

Attached Property Documentation

[read-only] Screen.logicalPixelDensity : real

每毫米逻辑象素的数目。这是平台提供的用于图像缩放计算的有效像素密度。

由于在Qt支持的不同平台上处理逻辑像素密度的方式不一致，建议使用物理像素(通过pixelDensity 属性)来实现可移植性。

这个QML属性是在Qt 5.1中引入的。

请参考 pixelDensity。
