Behavior QML 类型

为属性更改定义默认动画。 更多...

导入语句： import QtQuick 2.12
  • 所有成员的列表，包括继承成员

属性

  • animation : Animation
  • enabled : bool

详细信息

行为定义了在特定属性值更改时要应用的默认动画。

例如，下面的 Behavior 定义了一个NumberAnimation，它在Rectangle的width值发生变化时运行。当鼠标区域被点击，width改变，引发行为的动画：


  import QtQuick 2.0

  Rectangle {
      id: rect
      width: 100; height: 100
      color: "red"

      Behavior on width {
          NumberAnimation { duration: 1000 }
      }

      MouseArea {
          anchors.fill: parent
          onClicked: rect.width = 50
      }
  }

请注意，一个属性不能有多个分配的行为。要在一个 Behavior 中提供多个动画，请使用ParallelAnimation或SequentialAnimation。

如果状态变化有一个过渡，同样的属性相匹配的行为，在过渡动画覆盖该状态改变的行为。有关使用 Behaviors 为状态更改设置动画的一般建议，请参阅将 Qt 快速行为与状态一起使用。

请参阅 Qt Quick 中的动画和过渡，行为示例和Qt QML。

属性文档

[default] animation : Animation

此属性保存在触发行为时运行的动画。


enabled : bool

此属性保存当被跟踪属性更改值时是否会触发行为。

默认情况下启用行为。
