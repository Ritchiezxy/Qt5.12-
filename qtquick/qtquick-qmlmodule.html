Qt Quick中的QML类型

Qt Quick模块提供图形化的基本类型。只有当QML文档导入QtQuick名字空间时，这些类型才在QML文档中可用。

QtQuick模块的当前版本是2.11版本，因此可以通过以下语句导入:


  import QtQuick 2.11

有关Qt Quick的核心概念的更多信息，请访问QtQuick模块文档。

子模块

Qt Quick包含如下几个子模块：

  • XML List Model - 包含用于从XML数据创建模型的类型
  • Local Storage - 包含SQLite数据库的JavaScript接口的子模块
  • Particles - 为QML应用程序提供一个粒子系统
  • Window - 包含用于创建顶层窗口和访问屏幕信息的类型
  • Dialogs - 包含用于创建和与系统对话框交互的类型
  • Controls - 提供一组可重用的UI组件
  • Layouts - 包含用于在用户界面中排列项目的类型
  • Tests - 用于测试QML应用程序的类型。

基本类型

已经有很多基本类型由QML语言默认支持了。

除此之外，导入QtQuick后还会提供了以下基本类型:

color

ARGB颜色值。颜色类型引用ARGB颜色值。它可以通过以下几种方式指定:

font

具有QFont属性的字体值。字体类型引用具有QFont属性的字体值

matrix4x4

matrix4x4类型是一个4行4列的矩阵

quaternion

quaternion类型具有scalar, x, y, z属性（分量）

vector2d

vector2d类型具有x和y属性

vector3d

具有x、y和z属性的值

vector4d

vector4d类型具有x, y, z w四个属性
