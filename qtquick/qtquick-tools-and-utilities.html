Qt Quick 工具和实用程序

Qt 提供了多种工具和实用程序来实现丰富的开发人员体验，尤其是对于 Qt Quick 开发人员。以下部分简要介绍了这些工具和实用程序，并提供了指向有关它们的更多信息的链接。

Qt Quick Designer

Qt Quick Designer 可以使用大多数设计师熟悉的简单拖放手势来设计基于 Qt Quick 的 UI。它提供来自 Qt Quick 和 Qt Quick Controls 2 模块的 UI 元素，以及自定义 UI 元素的集成。

QML 调试器和分析器

作为一种声明性语言，一段 QML 代码提供了关于所定义实体的最少细节。在这种情况下，QML 调试器是一个非常有用的实用程序，它支持：

  • 调试 JavaScript 函数，
  • 执行 JavaScript 表达式，
  • 并检查 QML 属性。

除此之外，QML 分析器使您能够获得必要的诊断信息，允许您分析应用程序代码的性能问题。例如，每帧中的 JavaScript 过多、长时间运行的 C++ 函数等等。

QmlLive、GammaRay 和 Squish

QmlLive是一个第三方工具，它提供了一个 QML 运行时，能够实时呈现对代码的更改。它避免了在每次代码更改后重新构建应用程序并将其安装在目标设备上的需要。您还可以扩展它以构建适合您需求的自定义运行时。

GammaRay是一个有用的实用程序，可提供有关您的应用程序的诊断信息。它类似于前面部分中描述的 QML Profiler，但提供了更多功能。例如，创建的项目或 QObjects 的数量、进行的函数调用、每个函数调用所花费的时间、运行时的属性值自省等等。这些信息非常方便，尤其是在调试 QML 应用程序时。

Squish 是一个著名的测试工具，它通过记录您的操作或运行脚本来自动化 UI 测试。一旦设置了测试，UI 测试就更容易运行了。

Qt Creator

Qt Creator IDE 是增强使用 Qt Quick 的整体开发人员体验的关键工具。它的自动完成和调试功能使使用 Qt Quick 更加容易。除此之外，前面部分提到的大多数工具和实用程序都集成到其中，并且可以集成第三方工具，例如QmlLive和GammaRay。
