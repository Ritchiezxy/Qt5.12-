Turbulence(湍流） QML 类型

从噪声图像中提供类似流体的力。 更多...

导入语句： import QtQuick.Particles 2.12
继承自：

Affector

属性

  • noiseSource : url
  • strength : real

详细信息

流元素在其影响的区域上缩放噪声源，并使用该源的卷曲来生成力矢量。

湍流需要固定的大小。与其他影响器不同，0x0 湍流元素不会影响任何粒子。

来源应该是比较平滑的黑白噪声，比如柏林噪声。

属性文档

noiseSource : url

产生湍流的源图像。它将被缩放到元素的大小，因此相等或更大的大小将提供更好的结果。调整此图像是调整行为（例如漩涡在哪里或存在多少）的唯一方法。

来源应该是相对平滑的黑白噪声图像，例如柏林噪声。如果没有提供，将使用默认图像。


strength : real

任何一点的速度矢量的大小在零和二的平方根之间变化。然后将其乘以强度以获得受湍流影响的粒子的每秒速度。
