Qt Quick 示例 - 文本

这是与文本相关的 QML 示例集合。

文本是与文本相关的小型 QML 示例的集合。每个示例都是一个小的 QML 文件，通常包含或强调特定类型或特性。您可以运行并观察每个示例的行为。

Hello

Hello展示了如何更改Text类型的字母间距并为其设置动画。它使用顺序动画首先将 font.letterSpacing 属性设置0为503 秒以上，然后将文本移动到屏幕上的随机位置：


              SequentialAnimation on font.letterSpacing {
                  loops: Animation.Infinite;
                  NumberAnimation { from: 0; to: 50; easing.type: Easing.InQuad; duration: 3000 }
                  ScriptAction {
                      script: {
                          container.y = (screen.height / 4) + (Math.random() * screen.height / 2)
                          container.x = (screen.width / 4) + (Math.random() * screen.width / 2)
                      }
                  }
              }

字体

字体显示了使用Text类型的字体的不同方式。只需按名称，直接使用 font.family 属性：


              font.family: "Times"

或使用FontLoader类型：


      FontLoader { id: fixedFont; name: "Courier" }

或使用FontLoader并指定本地字体文件：


      FontLoader { id: localFont; source: "content/fonts/tarzeau_ocr_a.ttf" }

或者最后使用FontLoader并指定一个远程字体文件：


      FontLoader { id: webFont; source: "http://www.princexml.com/fonts/steffmann/Starburst.ttf" }

可用字体

可用字体展示了如何使用Qt全局对象和列表视图来显示系统上所有可用的字体。ListView类型使用可用的字体列表作为其模型：


          model: Qt.fontFamilies()

在委托内部，字体系列使用 modelData 设置：


                  font.family: modelData

横幅

Banner是一个简单的示例，展示了如何使用一行文本类型和NumberAnimation创建横幅。

图片标签

Img标记显示了使用 <img>标记在文本对象中显示图像的不同方法。

文本布局

文本布局展示了如何为文本项创建更复杂的布局。此示例使用 onLineLaidOut 处理程序将文本布置在两列中，该处理程序允许您定位和调整每行的大小：


          onLineLaidOut: {
              line.width = width / 2  - (margin)

              if (line.y + line.height >= height) {
                  line.y -= height - margin
                  line.x = width / 2 + margin
              }
          }
