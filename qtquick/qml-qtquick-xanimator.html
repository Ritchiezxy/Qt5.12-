XAnimator QML 类型

XAnimator类型使项目的x位置产生动画效果。 更多...

导入语句： import QtQuick 2.12
引入于： Qt 5.2
继承自：

Animator

  • 所有成员的列表，包括继承成员

详细信息

该Animator类型不同于普通的 Animation 类型。使用Animator时，动画可以在渲染线程中运行，动画完成时属性值会跳到最后。

Item::x的值在动画完成后更新。

以下代码段展示了如何将XAnimator与 Rectangle（矩形） 项一起使用。


  Rectangle {
      id: xmovingBox
      width: 50
      height: 50
      color: "lightsteelblue"
      XAnimator {
          target: xmovingBox;
          from: 10;
          to: 0;
          duration: 1000
          running: true
      }
  }

也可以使用on关键字将XAnimator直接绑定到 Item 实例。


  Rectangle {
      width: 50
      height: 50
      color: "lightsteelblue"
      XAnimator on x {
          from: 10;
          to: 0;
          duration: 1000
      }
  }
