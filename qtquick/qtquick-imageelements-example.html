Qt Quick 示例 - Image元素

这是与图像类型相关的 QML 示例集合。

Image Elements是与图像类型相关的小型 QML 示例的集合。有关更多信息，请访问用例 - QML 中的视觉元素。

运行该示例

要从 Qt Creator运行示例，请打开欢迎 模式并从 示例中选择示例。有关更多信息，请访问构建和运行示例。

阴影效果

Shadows展示了如何使用BorderImage为矩形项目创建阴影效果：


      BorderImage {
          anchors.fill: rectangle
          anchors { leftMargin: -6; topMargin: -6; rightMargin: -8; bottomMargin: -8 }
          border { left: 10; top: 10; right: 10; bottom: 10 }
          source: "shadow.png"
      }

使用 AnimatedSprite 的 Sprite 动画

AnimatedSprite展示了如何使用AnimatedSprite对象显示一个简单的动画：


      AnimatedSprite {
          id: sprite
          anchors.centerIn: parent
          source: "content/speaker.png"
          frameCount: 60
          frameSync: true
          frameWidth: 170
          frameHeight: 170
          loops: 3
      }

精灵动画将循环 3 次。

Sprite 动画与 SpriteSequence

SpriteSequence演示了使用精灵序列来绘制动画和交互式熊。SpriteSequence对象定义了五个不同的精灵。熊最初处于静止状态：


          Sprite{
              name: "still"
              source: "content/BearSheet.png"
              frameCount: 1
              frameWidth: 256
              frameHeight: 256
              frameDuration: 100
              to: {"still":1, "blink":0.1, "floating":0}
          }

单击场景时，动画将精灵序列设置为下降状态并为熊的 y 属性设置动画。


      SequentialAnimation {
          id: anim
          ScriptAction { script: image.goalSprite = "falling"; }
          NumberAnimation { target: image; property: "y"; to: 480; duration: 12000; }
          ScriptAction { script: {image.goalSprite = ""; image.jumpTo("still");} }
          PropertyAction { target: image; property: "y"; value: 0 }
      }

在动画结束时，熊被设置回其初始状态。
