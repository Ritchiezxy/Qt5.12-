ParticlePainter QML 类型

用于指定如何绘制粒子。 更多...

导入语句： import QtQuick.Particles 2.12
继承自：

Item

被继承：

CustomParticle, ImageParticle, and ItemParticle

  • 所有成员的列表，包括继承成员

属性

  • groups : list<string>
  • system : ParticleSystem

详细信息

默认实现不绘制任何内容。如果您想绘制可见的东西，请查看子类。

属性文档

groups : list<string>

将绘制哪些逻辑粒子组。

如果为空，它将绘制默认粒子组 ("")。


system : ParticleSystem

这是其粒子可以由元素绘制的系统。如果ParticlePainter是的直接子粒子系统，它会自动与它相关联。
