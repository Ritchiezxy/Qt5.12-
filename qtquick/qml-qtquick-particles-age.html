Age QML 类型

用于改变粒子年龄。 更多...

详细信息

年龄影响器允许您更改粒子在其生命周期中的位置。常见的用途是过早地使粒子失效，可能会给它们时间来制作动画。

年龄影响器有时也称为“杀死”影响器，因为使用默认参数，它会立即使所有受其影响的粒子失效。

年龄影响器仅适用于仍然存在的粒子。

属性文档

advancePosition : bool

AdvancePosition 确定位置、速度和加速度是否包含在影响器完成的模拟老化中。如果advancedPosition 为false，则位置、速度和加速度将保持不变，并且只有其他属性（例如不透明度）会在模拟中前进到粒子生命中那个点的正常位置。随着 AdvancePosition 设置为 true，位置、速度和加速度也将前进到粒子生命中那个点的正常位置，使其在屏幕上的位置前进。

Default value is true.


lifeLeft : int

设置粒子的生命量。受影响的粒子将前进到它们生命中的某个点，在那里它们将有这么多毫秒的生命。
