PathSvg QML 类型

使用SVG路径数据字符串定义路径。 更多...

导入语句： import QtQuick 2.12
  • 所有成员的列表，包括继承成员

属性

  • path : string

详细信息

下面的QML生成如下路径:


  Path {
      startX: 50; startY: 50
      PathSvg { path: "L 150 50 L 100 150 z" }
  }

注意: 并不总是支持将PathSvg与其他类型的元素混合。例如，当Shape由GL_NV_path_rendering支持时，一个ShapePath可以包含一个或多个PathSvg元素，或者一个或多个其他类型的元素，但不能同时包含两个元素。

请参考 Path, PathLine, PathQuad, PathCubic，PathArc，PathAngleArc 和 PathCurve。

属性文档

path : string

指定路径的SVG路径数据字符串。

有关这种格式的更多详细信息，请参阅W3C SVG路径数据。
