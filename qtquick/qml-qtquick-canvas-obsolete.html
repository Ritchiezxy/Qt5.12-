Canvas 的废弃成员

以下的 Canvas 成员已经废弃。 提供它们是为了让旧的源代码继续工作。我们强烈建议不要在新代码中使用它们。

属性

  • canvasWindow : rect （废弃）
  • tileSize : size （废弃）

属性文档

canvasWindow : rect

保存当前画布可见窗口。

默认情况下，canvasWindow大小与Canvas项目大小相同，且左上角点为(0,0)。

如果canvasSize与Canvas项目大小不同，Canvas项目可以通过改变Canvas的windowSize和/或位置来显示不同的可见区域。

请参考canvasSize and tileSize.


tileSize : size

保存画布渲染平铺的大小。

Canvas项目通过设置canvasSize, tileSize和canvasWindow进入平铺模式。这可以通过渲染和缓存tile而不是每次都渲染整个画布来提高渲染性能。

内存将只被当前可见区域内的那些块所消耗。

默认情况下，tileSize 与canvasSize相同。

请参考 canvasSize and canvasWindow.
