概念 - Qt Quick 中的视觉坐标

项坐标

Qt Quick 中使用的默认视觉坐标系统是项目坐标。这是一个笛卡尔坐标系，(0,0) 位于项目的左上角。x 轴向右增长，y 轴向下增长，因此项目的右下角位于坐标（宽度，高度）处。

单个项目的位置是根据其父项的坐标系指定的。这意味着从非同级项读取 x,y 值可能需要进行转换以将它们转换为相同的坐标系。发生这种情况时，场景坐标通常用作中间坐标系。

场景坐标

场景坐标是 (0,0) 对应于当前正在渲染场景的窗口左上角的坐标。场景坐标通常与窗口中根项的项坐标相同。

您可以使用您感兴趣的项目坐标系上的函数将项目转换为场景坐标。请参阅Item::mapFromItem和Item::mapToItem以转换为场景坐标或其他项目的坐标。

工作示例

下面的 QML 代码创建了一个正方形排列，并添加了点以识别点：


  Rectangle {
      width: 200
      height: 200
      color: "red"

      Rectangle {
          x: 100
          y: 100
          width: 100
          height: 100
          color: "blue"

          Rectangle {
              width: 50
              height: 50
              color: "green"
          }
      }
  }

在此图像中，黑点位于红色矩形项目坐标内的 (0,0) 处。如果红色矩形是场景的根项，那么黑点也将位于场景坐标中的 (0,0) 处。

蓝色矩形位于白点 (100,100) 处，相对于红色矩形的左上角。

绿色矩形没有指定 x,y，所以它的位置默认为 (0,0)。因为它位于其父级蓝色矩形的坐标中的 (0,0)，所以它位于该矩形的左上角。这与红色矩形坐标中 (100,100) 处的白点相同。
