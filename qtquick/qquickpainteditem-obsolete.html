QQuickPaintedItem 的废弃成员

以下的QQuickPaintedItem 成员已经废弃。 提供它们是为了让旧的源代码继续工作。我们强烈建议不要在新代码中使用它们。

属性

  • （废弃） contentsScale : qreal
  • （废弃） contentsSize : QSize

公用函数

（废弃） QRectF contentsBoundingRect() const
（废弃） void resetContentsSize()
  • 2个公用函数继承自 QObject

属性文档

contentsScale : qreal

废弃的内容缩放方法。

这个函数是为了兼容性而提供的，使用size()和textureSize()来决定你要绘制的尺寸。

访问函数：

qreal contentsScale() const
void setContentsScale(qreal)

通知信号：

void contentsScaleChanged()

请参考 width(), height(), 和 textureSize()。

contentsSize : QSize

用于设置内容大小的废弃方法。

这个函数是为了兼容性而提供的，使用size和textureSize来决定你要绘制的尺寸。

访问函数：

QSize contentsSize() const
void setContentsSize(const QSize &)

通知信号：

void contentsSizeChanged()

请参考 width(), height(), 和 textureSize()。

成员函数文档

QRectF QQuickPaintedItem::contentsBoundingRect() const

这个函数是为了兼容性而提供的，使用size和textureSize来决定你要绘制的尺寸。

请参考 width(), height(), 和 textureSize()。

void QQuickPaintedItem::resetContentsSize()

This convenience function is equivalent to calling setContentsSize(QSize()).
