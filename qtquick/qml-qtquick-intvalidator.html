详细信息

IntValidator类型提供了一种用于整数值的验证程序。

如果未设置语言环境，IntValidator使用默认语言环境来解释数字，并将接受语言环境特定的数字、组分隔符以及正负号。此外，IntValidator始终保证接受根据“C”语言环境格式化的数字。

属性文档

bottom : int

此属性保存验证器的最低可接受值。默认情况下，此属性的值源自可用的最低有符号整数（通常为 -2147483647）。


locale : string

此属性保存用于解释数字的语言环境的名称。

请参考 Qt.locale()。


top : int

此属性包含验证器可接受的最高值。默认情况下，此属性的值源自可用的最高有符号整数（通常为 2147483647）。
