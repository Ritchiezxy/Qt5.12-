RegExpValidator QML 类型

提供一个字符串验证器。 更多...

属性

  • regExp : regExp

详细信息

RegExpValidator类型提供了一个验证器，它将匹配指定正则表达式的任何字符串视为有效。

属性文档

regExp : regExp

此属性保存用于验证的正则表达式。

注意，这个属性在JS语法中应该是一个正则表达式，例如/a/用于匹配"a"的正则表达式。

默认情况下，该属性包含一个正则表达式，pattern .*匹配任何字符串。

下面你可以找到一个指定了RegExpValidator的TextInput对象的例子:


  TextInput {
      id: hexNumber
      validator: RegExpValidator { regExp: /[0-9A-F]+/ }
  }

更多正则表达式的例子:

  • 一组用逗号隔开的一到三个位置的数字:
    
  /\d{1,3}(?:,\d{1,3})+$/
  • 小数点前最多由3位组成，小数点后最多由1至2位组成:
    
  /(\d{1,3})([.,]\d{1,2})?$/
