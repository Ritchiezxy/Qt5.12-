详细信息

CanvasImageData对象保存图像的像素数据。

CanvasImageData对象具有存储在该对象中的数据的实际维数，并保存了以RGBA顺序包含数据的一维数组，作为范围为0到255的整数。

请参考 width, height, data，Context2D::createImageData()，Context2D::getImageData() 和 Context2D::putImageData()。

属性文档

data : object

保存以RGBA顺序包含数据的一维数组，作为范围为0到255的整数。


height : int

保存ImageData对象中数据的实际高度维度(以设备像素为单位)。


width : int

保存ImageData对象中数据的实际宽度维数(以设备像素为单位)。
