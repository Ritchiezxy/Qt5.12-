Qt Quick文本输入处理和验证器

文本可视化类型

Qt Quick提供了几种在屏幕上显示文本的类型。Text类型将在屏幕上显示格式化的文本，TextEdit类型将在屏幕上放置多行编辑器，而TextInput将在屏幕上放置单个可编辑行字段。

要了解更多关于它们特定特性和属性的信息，请访问它们各自的文档。

验证输入文本

验证器类型强制执行TextInput对象的类型和格式。

Qt Quick文本验证器

验证文本输入的类型


  Column {
      spacing: 10

      Text {
          text: "Enter a value from 0 to 2000"
      }
      TextInput {
          focus: true
          validator: IntValidator { bottom:0; top: 2000}
      }
  }

验证器类型绑定到TextInput的validator属性。


  Column {
      spacing: 10

      Text {
          text: "Which basket?"
      }
      TextInput {
          focus: true
          validator: RegExpValidator { regExp: /fruit basket/ }
      }
  }

片段中的正则表达式将只允许输入的文本为fruit basket.

请注意，QML 解析 JavaScript 正则表达式，而 Qt 的QRegExp类的正则表达式基于 Perl 正则表达式。
