ParallelAnimation QML 类型

使动画能够并行运行。 更多...

导入语句： import QtQuick 2.12
继承自：

Animation

  • 所有成员的列表，包括继承成员

详细信息

SequentialAnimation和ParallelAnimation类型允许多个动画一起运行。SequentialAnimation中定义的动画一个接一个运行，而ParallelAnimation中定义的动画同时运行。

以下动画并行运行两个数字动画。该矩形由动画其前进到（50,50）x和y在同一时间属性。


  import QtQuick 2.0

  Rectangle {
      id: rect
      width: 100; height: 100
      color: "red"

      ParallelAnimation {
          running: true
          NumberAnimation { target: rect; property: "x"; to: 50; duration: 1000 }
          NumberAnimation { target: rect; property: "y"; to: 50; duration: 1000 }
      }
  }

与任何其他动画类型一样，ParallelAnimation可以通过多种方式应用，包括过渡、行为和属性值源。Qt Quick文档中的动画和转换展示了多种创建动画的方法。

注意：一旦动画被分组为SequentialAnimation或ParallelAnimation，就不能单独启动和停止；在SequentialAnimation或ParallelAnimation必须启动并停止为一组。

请参考 SequentialAnimation，Qt中的动画和过渡 和 Qt Quick 示例 - 动画。
