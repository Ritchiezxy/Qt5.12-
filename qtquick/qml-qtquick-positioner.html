Positioner QML 类型

提供附加属性，其中包含有关项目在定位器中的位置的详细信息。更多...

导入语句： import QtQuick 2.12
  • 所有成员的列表，包括继承成员

Attached Properties

  • index : int
  • isFirstItem : bool
  • isLastItem : bool

详细信息

定位器类型的对象附加到Column, Row、 Flow 或Grid中的顶级子项。它提供的属性允许子项确定它在其父Column, Row、 Flow 或Grid的布局中的位置。

例如，下面是通过Repeater创建的具有 16 个子矩形的Grid。每个矩形使用显示在网格中的指数Positioner.index，并且第一个项目是通过采取了不同的色彩Positioner.isFirstItem考虑：


  Grid {
      Repeater {
          model: 16

          Rectangle {
              id: rect
              width: 30; height: 30
              border.width: 1
              color: Positioner.isFirstItem ? "yellow" : "lightsteelblue"

              Text { text: rect.Positioner.index }
          }
      }
  }

附加属性文档

Positioner.index : int

此属性允许项目在定位器中确定其索引。


Positioner.isFirstItem : bool

这些属性允许项目分别确定它是定位器中的第一个项目还是最后一个项目。


Positioner.isLastItem : bool

这些属性允许项目分别确定它是定位器中的第一个项目还是最后一个项目。
