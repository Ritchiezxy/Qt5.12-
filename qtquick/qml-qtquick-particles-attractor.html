Attractor（吸引器） QML 类型

用于将粒子吸引到特定点。 更多...

导入语句： import QtQuick.Particles 2.12
继承自：

Affector

  • 所有成员的列表，包括继承成员

属性

  • affectedParameter : AffectableParameter
  • proportionalToDistance : Proportion

详细信息

请注意，此元素的大小和位置会影响它影响的粒子。吸引到的点的大小始终为 0x0，该点的位置由 pointX 和 pointY 属性指定。

请注意，Attractor 具有标准的 Item x、y、width 和 height 属性。像其他影响因素一样，这些代表受影响的区域。它们不代表作为景点目标的 0x0 点。

属性文档

affectedParameter : AffectableParameter

直接影响粒子的属性。

  • Attractor.Position(位置）
  • Attractor.Velocity（速度）
  • Attractor.Acceleration（加速度）

proportionalToDistance : Proportion

从粒子到点的距离如何影响吸引力的强度。

  • Attractor.Constant（常数）
  • Attractor.Linear（线性）
  • Attractor.InverseLinear（逆线性）
  • Attractor.Quadratic（二次方）
  • Attractor.InverseQuadratic（逆二次方）
