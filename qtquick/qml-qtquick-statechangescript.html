StateChangeScript QML 类型

指定如何在某个状态下运行脚本。 更多...

导入语句： import QtQuick 2.12
  • 所有成员的列表，包括继承成员

属性

  • name : string
  • script : script

详细信息

StateChangeScript在进入状态时运行。您可以选择使用ScriptAction来指定StateChangeScript应该运行的过渡点。


  State {
      name: "state1"
      StateChangeScript {
          name: "myScript"
          script: doStateStuff();
      }
      // ...
  }
  // ...
  Transition {
      to: "state1"
      SequentialAnimation {
          NumberAnimation { /* ... */ }
          ScriptAction { scriptName: "myScript" }
          NumberAnimation { /* ... */ }
      }
  }

请参考 ScriptAction。

属性文档

name : string

该属性保存脚本的名称。ScriptAction可以使用此名称来定位特定脚本。

请参考 ScriptAction::scriptName。


script : script

此属性保存状态为当前时要运行的脚本。
