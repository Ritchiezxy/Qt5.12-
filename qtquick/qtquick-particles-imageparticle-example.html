Qt Quick 粒子示例 - 图像粒子

这是在 QML 粒子系统中使用 Affector 的示例集合。

这是与在粒子系统中使用影响器有关的小型 QML 示例的集合。每个示例都是强调特定类型或功能的小型 QML 文件。

所有这些都同时展示了ImageParticle的几个特性。


  sprites: [
      Sprite {
          name: "bear"
          source: "../../images/bear_tiles.png"
          frameCount: 13
          frameDuration: 120
      }
  ]
  colorVariation: 0.5
  rotationVelocityVariation: 360
  colorTable: "../../images/colortable.png"

Colored 显示一个带有一些颜色变化的简单ImageParticle 。


  ImageParticle {
      anchors.fill: parent
      source: "qrc:///particleresources/star.png"
      alpha: 0
      alphaVariation: 0.2
      colorVariation: 1.0
  }

颜色表设置粒子生命周期的颜色以提供固定的彩虹效果。


  source: "qrc:///particleresources/glowdot.png"
  colorTable: "../../images/colortable.png"
  sizeTable: "../../images/colortable.png"

变形旋转并挤压海星粒子。


  ImageParticle {
      system: sys
      groups: ["goingLeft", "goingRight"]
      source: "../../images/starfish_4.png"
      rotation: 90
      rotationVelocity: 90
      autoRotation: true
  }
  ImageParticle {
      system: sys
      groups: ["goingDown"]
      source: "../../images/starfish_0.png"
      rotation: 180
      yVector: PointDirection { y: 0.5; yVariation: 0.25; xVariation: 0.25; }
  }

Rotation 演示了 autoRotate 属性，以便粒子沿它们行进的方向旋转。

共享演示了当多个ImageParticles尝试渲染同一个粒子时会发生什么。下面的ImageParticle在ListView中渲染粒子:


  ImageParticle {
      anchors.fill: parent
      system: particles
      source: "../../images/flower.png"
      alpha: 0.1
      color: "white"
      rotationVariation: 180
      z: -1
  }

下面的ImageParticle放置在列表高亮中，并将粒子呈现在另一个ImageParticle 之上。


  ImageParticle {
      anchors.fill: parent
      system: particles
      source: "../../images/flower.png"
      color: "red"
      clip: true
      alpha: 1.0
  }

请注意，因为它在此ImageParticle中设置了颜色和 alpha ，所以它以不同的颜色渲染粒子。由于它没有指定任何关于旋转的内容，因此它与另一个ImageParticle共享旋转，以便花朵在两者中以相同的方式旋转。请注意，您可以在另一个ImageParticle中撤消旋转，您只需将 rotationVariation 显式设置为 0。

Sprites 演示了使用图像粒子来渲染动画精灵而不是每个粒子的静态图像。
