OpacityAnimator（透明动画师） QML 类型

OpacityAnimator类型的动画项的不透明度。 更多...

导入语句： import QtQuick 2.12
Since: Qt 5.2
继承自：

Animator

  • 所有成员的列表，包括继承成员

详细信息

Animator类型不同于普通的动画类型。当使用Animator时，动画可以在渲染线程中运行，当动画完成时，属性值将跳转到结束。

Item::opacity的值在动画完成后更新。

以下代码段展示了如何将OpacityAnimator与 Rectangle（矩形） 项一起使用。


  Rectangle {
      id: opacityBox
      width: 50
      height: 50
      color: "lightsteelblue"
      OpacityAnimator {
          target: opacityBox;
          from: 0;
          to: 1;
          duration: 1000
          running: true
      }
  }

也可以使用on关键字将OpacityAnimator直接绑定到 Item 实例。


  Rectangle {
      width: 50
      height: 50
      color: "lightsteelblue"
      OpacityAnimator on opacity{
          from: 0;
          to: 1;
          duration: 1000
      }
  }
